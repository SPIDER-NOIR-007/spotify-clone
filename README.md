# 🎵 Spotify Clone

A fully functional Spotify-inspired music player built with vanilla HTML, CSS, and JavaScript.

🔗 **Live Demo:** [spotify-clone-six-nu-54.vercel.app](https://spotify-clone-six-nu-54.vercel.app/)

---

## 📸 Preview

> A sleek, dark-themed music player replicating the core Spotify experience — browse albums, play songs, and control playback — all in the browser.

---

## ✨ Features

- 🎧 **Music Playback** — Play, pause, skip to next/previous tracks
- 📁 **Album Browsing** — Dynamically loaded album cards with cover art
- 🔊 **Volume Control** — Adjustable volume slider
- ⏩ **Seek Bar** — Click to jump to any point in a song
- 🕒 **Timestamps** — Live current time and total duration display
- 📋 **Song Queue** — View and select songs within an album
- 📱 **Responsive Layout** — Works across different screen sizes

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| **HTML5** | Structure and semantic markup |
| **CSS3** | Styling, Flexbox layout, animations |
| **JavaScript (Vanilla)** | DOM manipulation, audio API, dynamic rendering |

> No frameworks. No libraries. Pure browser APIs.

## 🚀 Getting Started

### Prerequisites

- A modern web browser
- (Optional) [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension for VS Code

### Run Locally

```bash
# Clone the repository
git clone https://github.com/bhavyvaghani829-beep/spotify-clone.git

# Navigate into the project
cd spotify-clone

# Open index.html directly, or use Live Server in VS Code
```

---

## 🎮 How to Use

1. Open the app — albums load automatically from the `songs/` directory
2. Click an album card to expand its song list
3. Click any song to start playback
4. Use the bottom player bar to:
   - ▶️ Play / ⏸ Pause
   - ⏮ Previous / ⏭ Next track
   - 🔊 Adjust volume
   - 🖱️ Seek through the track

## 🌐 Deployment

This project is deployed on **Vercel**.

To deploy your own fork:

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project**
3. Import your GitHub repository
4. Click **Deploy** — done!
